## Задача
1. Нажать “Do one!” на этом сайте: https://www.architecturalkatas.com
2. Ознакомиться с вводными по вашему кейсу
3. Целиком пройти этап “Сбор и анализ требований”
    - Отсев
    - Scope Refinement
    - Функциональные требования
    - Нефункциональные требования
    - Ограничения
    - Расчет нагрузки
4. Сформировать отчет по этим пунктам в любом читаемом формате (желательно .md)
5. Доп. вводные: вы находитесь в России, 2026 год

## Как сдавать
1. Сделайте Fork этого репозитория на Github
2. Сделайте ветку "HW1" в своем репозитории
3. Сделайте ваш репозиторий приватным
4. Разместите отчет ДЗ №1 в директории "2"
5. Сделайте Pull Request в ветку main
6. Добавьте пользователя nikolaysavelev в репозиторий и в reviewer PR
7. Напишите в телеграме в ЛС @nikolaysavelev о том, что сделали ДЗ и можно проверять
